GraviSynth

Where code meets sound

A fully modular digital synthesizer built with C++ and JUCE. Features a visual node-based graph editor, core modules (Oscillator, Filter, VCA, ADSR, LFO, Sequencer), FX modules (Delay, Distortion, Reverb), 8-voice polyphony, and an AI Sound Designer that generates patches from natural language descriptions.
